# Examples - 示例项目

## 模块概述

Examples模块包含了OSATE2的示例AADL模型和项目，用于演示AADL建模能力、OSATE工具功能，以及帮助用户学习和理解AADL语言和OSATE工具的使用。

## 主要组件

### 示例包
- **org.osate.examples** - 核心示例集合
  - 基础AADL示例
  - 功能演示示例
  - 最佳实践示例
  - 应用领域示例

### 特性定义
- **org.osate.examples.feature** - 示例特性定义
  - 示例安装包
  - 依赖关系
  - 分类信息

### Releng支持
- **examples.releng** - 示例发布工程
  - 构建配置
  - 打包定义

## 示例分类

### 1. 基础示例
**目的**：介绍AADL基本概念

**内容**：
- **简单组件** - 进程、线程、数据的基本定义
- **端口和连接** - 数据端口、事件端口、事件数据端口
- **特性组** - 特性分组的使用
- **子组件** - 组件层次结构
- **模式** - 操作模式定义和切换

**学习目标**：
- 理解AADL基本语法
- 掌握组件类型和实现
- 学习端口和连接机制
- 了解层次化建模

### 2. 属性示例
**目的**：演示AADL属性使用

**内容**：
- **预定义属性** - 标准属性集的使用
- **自定义属性** - 用户定义属性
- **属性继承** - 属性值继承和覆盖
- **属性关联** - 属性与组件的关联

**涉及属性**：
- Timing Properties (Period, Deadline, WCET)
- Memory Properties (Stack_Size, Heap_Size)
- Thread Properties (Dispatch_Protocol, Priority)
- Communication Properties (Latency, Bandwidth)

### 3. 分析示例
**目的**：演示各种分析功能

**内容**：
- **流延迟分析** - 端到端流延迟计算
- **资源预算分析** - CPU、内存、带宽预算
- **调度分析** - 任务调度可行性
- **架构分析** - 系统一致性检查

**分析场景**：
- 实时性能验证
- 资源分配检查
- 系统集成验证

### 4. 错误模型示例
**目的**：演示错误建模和分析

**内容**：
- **错误类型定义** - 错误类型库
- **错误行为** - 状态机定义
- **错误传播** - 错误传播路径
- **故障树分析** - FTA生成和分析
- **组合错误行为** - 复杂系统的错误行为

**应用场景**：
- 安全性分析
- 可靠性评估
- 故障诊断

### 5. 行为附件示例
**目的**：演示行为建模

**内容**：
- **状态机** - 简单和复杂状态机
- **动作序列** - 计算和通信动作
- **时间约束** - 执行时间和截止时间
- **条件和循环** - 控制流结构

**应用**：
- 线程行为规范
- 协议实现
- 控制逻辑

### 6. ALISA示例
**目的**：演示需求和验证

**内容**：
- **需求定义** - ReqSpec示例
- **验证计划** - Verify示例
- **保证案例** - Assure示例
- **完整工作流** - 从需求到验证的完整流程

**工作流**：
- 需求追踪
- 自动验证
- 证据收集
- 合规性报告

## 示例结构

### 典型项目结构
```
example-project/
├── packages/           # AADL包
│   ├── components.aadl
│   ├── system.aadl
│   └── properties.aadl
├── instances/          # 实例模型
│   └── system_instance.aaxl2
├── diagrams/           # 图形化视图
│   └── system.aadl_diagram
├── requirements/       # 需求文档(ALISA)
│   └── requirements.reqspec
├── verify/            # 验证计划(ALISA)
│   └── verification.verify
└── README.md          # 项目说明
```

### 文档标准
每个示例应包含：
- **README** - 项目概述和学习目标
- **注释** - 详细的代码注释
- **参考资料** - 相关标准和文档链接

## 使用方法

### 导入示例
1. **通过Eclipse**
   - File → Import → OSATE → Import Examples
   - 选择感兴趣的示例
   - 导入到工作空间

2. **手动导入**
   - 复制示例项目
   - Import → Existing Projects
   - 构建项目

### 运行分析
1. 实例化系统模型
2. 在实例上运行分析
3. 查看分析结果
4. 理解结果含义

- 添加新组件扩展系统
- 尝试不同的设计方案
- 比较分析结果
